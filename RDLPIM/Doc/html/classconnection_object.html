connectionObject Class Reference

Detailed Description

The connectionObject is a wrapper around the window SOCKET class.

Warning
This is not platform independant although should be in the future
Author
Guy Collins

Example

connectionObject connection;
connection.setIP("127.0.0.1);
connection.setPort(8000);
connection.connectToServer();
connection.setNonBlocking(true);
Buffer buff;
while(true){
if(connection.canRead()){
connection.recieve(&buff);
buff.fullPrint();
} else {
Sleep(1000);
}
}

Method to put the connectionObject into a connection accept state.

int connectionObject::connectToServer ( )

Method to connect socket to an endpoint.

The Endpoint is the IP address defined by setIP() and port defined by setPort(). If successful, the connection is bound to the client SOCKET object else a relevant warning is pipped to console.

Returns
status code. 0 no errors, -1 endpoint is not defined, -2 WSA error occured.
