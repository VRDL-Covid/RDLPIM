DataElement Class Reference

Detailed Description

A class to handle the data transfer of complex and primitive objects into and out of the DataBase instance.

the serialised format is as follows:

{[char]=[char]=[int][char]}

"{VarName=VarType=Bytes Value}" (no space delimeter between Bytes and Value since Bytes is always 4Bytes in size and describes the length of Value)

Overloaded functions have been provided to allow primitives to be injected into DataElements readily

Example

DataElement de("varName");
double value;
Buffer serialised;
value = 3.14159;
de.set(value);
serialised = de.Serialise();
serialised.fullPrint();
// { int = 8 bytes } { binary for 3.14159 as double }
//"{varName=double=0x08 0x00 0x00 0x00 0x40 0x09 0x21 0xF9 0xF0 0x1B 0x86 0x6E}"
